# TBL331 Veritabanı Yönetim Sistemleri Dönem Projesi

Bu depo, TBL331 Veritabanı Yönetim Sistemleri dersi dönem projesi kapsamında Grup 142 tarafından geliştirilen veritabanı yönetim sistemini içermektedir.

## Proje Özeti
Şirket içi çalışanların, departmanların, izinlerin, performans değerlendirmelerinin ve maaş geçmişinin (bordro) yönetimini sağlayan bir veritabanı tasarımıdır.

## Geliştirme Ortamı
* **Veritabanı Yönetim Sistemi:** PostgreSQL 18.3

## Kurulum ve Çalıştırma
Proje kapsamında yazılan SQL sorgularını çalıştırmak için aşağıdaki adımları takip edebilirsiniz:
1. PostgreSQL veritabanı sunucunuza bağlanın.
2. Yeni bir veritabanı oluşturun.
3. `grup142_sql_betikleri.txt` dosyası içerisindeki SQL betiklerini bu veritabanında çalıştırın.
